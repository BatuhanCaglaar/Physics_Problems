# Problem 6: Parameters of a Sinusoidal Wave

**Given Equation:**
$$y(x,t) = 0.05 \sin(2\pi x - 50\pi t)$$

---

## Solution:
Comparing with the standard wave equation $y(x,t) = A \sin(kx - \omega t)$:

**a) Amplitude (A):** The coefficient of the sine function.
**$A = 0.05 \, \text{m}$**

**b) Wavelength ($\lambda$):** From the wave number $k = 2\pi$.
$$k = \frac{2\pi}{\lambda} \implies 2\pi = \frac{2\pi}{\lambda} \implies \mathbf{\lambda = 1 \, \text{m}}$$

**c) Frequency (f):** From the angular frequency $\omega = 50\pi$.
$$\omega = 2\pi f \implies 50\pi = 2\pi f \implies \mathbf{f = 25 \, \text{Hz}}$$

**d) Wave Speed (v):**
$$v = f \cdot \lambda = 25 \times 1 = \mathbf{25 \, \text{m/s}}$$
