# Problem 7: Wavelength of Higher Harmonics

**Given:**
- String length: $L = 80 \, \text{cm} = 0.8 \, \text{m}$
- Number of antinodes: $n = 4$

---

## Solution:
For a string fixed at both ends, the number of antinodes corresponds to the $n$-th harmonic. The relationship between length and wavelength is:
$$L = n \left( \frac{\lambda}{2} \right)$$

Substituting the known values:
$$0.8 = 4 \left( \frac{\lambda}{2} \right)$$
$$0.8 = 2\lambda \implies \mathbf{\lambda = 0.4 \, \text{m}}$$

**Answer:** The wavelength of the wave is **0.4 m** (or 40 cm).
