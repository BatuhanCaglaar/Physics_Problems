# Problem 4: Phase Difference Analysis

**Given:**
- Path difference: $\Delta x = \lambda/3$

---

## Solution:
The phase difference ($\Delta \phi$) in radians is related to the path difference ($\Delta x$) by the formula:
$$\Delta \phi = \frac{2\pi}{\lambda} \Delta x$$

Substituting the given value:
$$\Delta \phi = \frac{2\pi}{\lambda} \cdot \frac{\lambda}{3} = \mathbf{\frac{2\pi}{3} \, \text{rad}}$$

**Answer:** The phase difference is **$2\pi/3$ radians** (equivalent to 120°).
