# Problem 2: Wave Speed on a Guitar String

**Given:**
- String length: $L = 64 \, \text{cm} = 0.64 \, \text{m}$
- Fundamental frequency (1st harmonic): $f_1 = 330 \, \text{Hz}$

---

## Solution:
For a string fixed at both ends, the fundamental frequency (one antinode) occurs when the length of the string is half of the wavelength:
$$L = \frac{\lambda}{2} \implies \lambda = 2L$$
$$\lambda = 2 \times 0.64 = 1.28 \, \text{m}$$

Now, calculate the wave speed ($v$):
$$v = f \cdot \lambda$$
$$v = 330 \times 1.28 = \mathbf{422.4 \, \text{m/s}}$$

**Answer:** The speed of the wave on this string is **422.4 m/s**.
