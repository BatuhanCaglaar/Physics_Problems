# Problem 8: Electromagnetic Spectrum Classification

**Requirement:** List the radiation types in order of **increasing wavelength** (from the shortest wavelength/highest energy to the longest wavelength/lowest energy).

---

## Solution:
The electromagnetic spectrum, arranged from the shortest to the longest wavelength, is as follows:

1. **Gamma rays** (Shortest wavelength, highest frequency)
2. **X-rays**
3. **Ultraviolet**
4. **Infrared**
5. **Microwaves**
6. **Radio waves** (Longest wavelength, lowest frequency)

**Answer:**
Gamma rays < X-rays < Ultraviolet < Infrared < Microwaves < Radio waves.
