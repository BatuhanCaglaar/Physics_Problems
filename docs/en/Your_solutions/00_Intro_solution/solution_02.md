# Solution 2: Systems of Equations

To solve this system, I am using the **Substitution Method**, starting with the simpler second equation.

### 1. Isolate $x$
From the second equation ($x - y = 1$):
$$x = y + 1$$

### 2. Substitute into the first equation
Plug the expression for $x$ into the first equation ($2x + 3y = 12$):
$$2(y + 1) + 3y = 12$$

### 3. Solve for $y$
Expand and combine terms:
$$2y + 2 + 3y = 12$$
$$5y = 10 \implies y = 2$$

### 4. Find the value of $x$
Using $x = y + 1$:
$$x = 2 + 1 = 3$$

**Final Result:**
The solution is **$x = 3, \quad y = 2$**.
